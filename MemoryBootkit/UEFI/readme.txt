AaLl86 EFI EDK Application model
Last revision: 06/01/2013

This model is compatible with Visual Studio 2008 and let you been able to develop and compile your EFI startup application
(and even EFI driver) with the handy Visual Studio 2008 environment.
Intel EFI EDK must be installed and environment variable "EDK_SOURCE" MUST be set to point to EDK root directory.
